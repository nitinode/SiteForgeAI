# SiteForgeAI

SiteForgeAI is an AI-powered website builder that allows users to generate and deploy websites using AI prompts.

## Features

- AI website generation
- Live website preview
- User authentication
- Stripe subscription system
- Dashboard for managing generated sites

## Tech Stack

Frontend:
- React
- Vite
- Redux

Backend:
- Node.js
- Express.js
- MongoDB

Services:
- OpenRouter AI
- Stripe Billing
- Firebase Authentication

## Installation

Clone the repository

git clone https://github.com/nitinode/SiteForgeAI.git

### Client

cd client  
npm install  
npm run dev

### Server

cd server  
npm install  
npm run dev
